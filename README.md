# Projeto de ITP - Sistema de Gerenciammento de Banco de dados

## Contribuidores
`Felipe Baraúna`

## Requisitos
- [git](https://git-scm.com/) 
- [gcc](https://gcc.gnu.org/)

## Instalação

Clone o repositorio
```bash
git clone https://github.com/shgates/db.git
```

Vá para o diretório criado e compile o projeto
```bash
cd db
make
```

Rode o projeto
```bash
./db
```
